# js-04-cadastro-cartao

#  Página de Cadastro de Cartão

Aplicação web interativa que simula o cadastro de um cartão de crédito com atualização dinâmica em tempo real.

O projeto reproduz visualmente um cartão físico com efeito 3D e atualiza automaticamente as informações conforme o usuário preenche o formulário.


# Funcionalidades

- Formatação automática do número do cartão (#### #### #### ####)
- Nome do titular convertido automaticamente para maiúsculo
- Formatação automática da validade (MM/AA)
- Exibição do CVC no verso do cartão
- Efeito 3D de rotação ao focar no campo CVC
- Detecção automática de bandeira (Visa e Mastercard)
- Interface responsiva
- Simulação visual com chip e símbolo de pagamento por aproximação


# Tecnologias Utilizadas

- HTML5  
- CSS3 (Flexbox, Gradientes, Transformações 3D)  
- JavaScript (Vanilla JS)  
- Expressões Regulares (Regex)


# Conceitos Aplicados

Durante o desenvolvimento foram praticados:

- Manipulação do DOM
- Tratamento de eventos (`input`, `focus`, `blur`)
- Formatação dinâmica de dados com Regex
- Lógica condicional para identificação de bandeira (padrão BIN)
- CSS 3D (`perspective`, `rotateY`, `backface-visibility`)
- Organização e separação de responsabilidades no código


# Recursos Visuais

- Ilustração de chip de cartão
- Ícone de pagamento por aproximação
- Logotipos das bandeiras Visa e Mastercard
